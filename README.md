# 同人印刷シミュレーター

## 推奨環境

手元の環境。多分バージョン違くても動く
node v12.11.1
yarn 1.19.0

## インストール手順

```
$ yarn install
```

## 起動方法

```
node app.js
```

`localhost:3000` にアクセスして、ページが表示されればおk

## 構成

表示されるページは `views/index.html` 内にあります。
勉強用ということでほぼほぼ全てのコードが上記ファイル内に書かれていますが、
価格表のCSVデータなどは別ファイルに書き出して読み込んでいます。
これらは `public/` 以下に保存されています。
